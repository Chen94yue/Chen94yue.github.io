1. 按文件名查找文件
```
find . -name "name"
```
.：当前文件夹及其子目录
"name"：文件名，可用正则表达式

2. 压缩文件为zip格式
```
zip -r myfile.zip ./* 
```

3. 查看当前文件夹下文件的大小
```
du -h --max-depth=1
```
